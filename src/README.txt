Lol Nexus 

Funcionalidades Clave del proyecto

1. Listado de Personajes:

    - Mostrar una lista de todos los personajes jugables con sus imágenes y nombres.

2. Detalles del Personaje:

    - Al hacer clic en un personaje, mostrar detalles como habilidades, estadísticas y roles.

3. Comparación de Personajes:

    - Permitir a los usuarios comparar las estadísticas y habilidades de dos o más personajes.

4. Filtros y Búsqueda:

    - Implementar una barra de búsqueda y filtros para encontrar personajes específicos fácilmente.

5. Gráficos y Visualizaciones:

    - Usar gráficos para mostrar visualmente las estadísticas y roles de los personajes.
